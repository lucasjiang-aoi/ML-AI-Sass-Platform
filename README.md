# 🚀 ML-AI-SaaS-Platform

A modern **AI/ML SaaS platform** built for scalable, production-ready machine learning applications and intelligent automation workflows.

Designed with a cloud-first architecture, this platform enables developers and businesses to integrate AI capabilities into real-world products with speed, flexibility, and scalability.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/lucasjiang-aoi/ML-AI-Sass-Platform.git
cd ML-AI-Sass-Platform
```

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

---

## ⚙️ Environment Variables

Create a `.env` file in the root directory:

```env
OPENAI_API_KEY=your_api_key
DATABASE_URL=your_database_url
JWT_SECRET=your_secret
```
